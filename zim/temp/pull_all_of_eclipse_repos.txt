====== pull all of eclipse repos ======
Created Thursday 23 March 2017

wget -r -p --no-parent https://git.eclipse.org/c/

**Get all http clone links:**
grep -r https | grep clone | cut -f5 -d " " | cut -f1 -d "<"

http clone links with git clone command:
grep -r "http*://git.*git" | g clone | cut -f4 -d "=" | cut -f2 -d "'" | { while read i; do echo git clone $i; done }


**Get all https clone links:**
grep -r "http*://git.*git" | g clone | cut -f4 -d "=" | cut -f2 -d "'"


**Combined to file & tee**
grep -r https | grep clone | cut -f5 -d " " | cut -f1 -d "<" | tee ~/tmp2/gitrepostoclone && grep -r "http*://git.*git" | g clone | cut -f4 -d "=" | cut -f2 -d "'" | tee -a ~/tmp2/gitrepostoclone

Combined, but append to existing.
grep -r https | grep clone | cut -f5 -d " " | cut -f1 -d "<" | tee -a ~/tmp2/gitrepostoclone && grep -r "http*://git.*git" | g clone | cut -f4 -d "=" | cut -f2 -d "'" | tee -a ~/tmp2/gitrepostoclone


**Grep out BrowserFunction return value.**
grep -r -A 25 "extends BrowserFunction" | g -v "return null" | g -B 25 "return.*;"
^ Maybe change 25 to a higher number... but need to get just the class somehow.
