===== Sudo without password =====
1) Edit:
	sudo vim [[/etc/sudoers]]
	Uncomment:
	## Same thing without a password
	#%wheel  ALL=(ALL)       NOPASSWD: ALL     <<< this line uncomment
2) Add yourself to wheel:
	usermod your_login -a -G wheel
3) Logout and log back in.
*tested, works F24.*

==== VPN setup ====
* **VPN**: https://mojo.redhat.com/docs/DOC-4393
	sudo dnf install vpnc
	sudo vi [[/etc/vpnc.conf]]
**Paste:**
IPSec gateway 66.187.233.55 
IKE Authmode psk  
IPSec ID RH-standard  
IPSec secret nodnerip  
IKE DH Group dh2  
NAT Traversal Mode natt  
Xauth username lufimtse

sudo chmod +x [[/etc/vpnc/vpnc-script]]

(The enable ispec doesn't seem to be relevant anymore)
**cmds:**
vpnc  & vpnc-disconnect
