====== Data types ======
Created Thursday 03 November 2016

char = CHAR_BITS = 8 bits on POSIX. Can be >= 8 in C on some systems. ([[http://stackoverflow.com/questions/3200954/what-is-char-bit|reference]])


**VS Java:**
byte = 8 bits. (-128 to 127)

C char == Java byte
