====== 17·08·18 vim-x11 has clipboard ======
Created Friday 18 August 2017
@blog @vim 

The vim-X11 package provides vimx, which gives me access to system clipboard.

With the setting:
set clipboard=unnamedplus

I can copy things to the system clipboard instead of only interneal clipboard. So cool.
