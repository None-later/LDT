====== date ======
Created Wednesday 29 June 2016

Date:
	User friendly:
	date +'%Y-%m(%b)-%d(%a)'
	> 2016-06(Jun)-29(Wed)

	Weeknumbers (%V) and file system friendly:
	date +'%Y-W%V-%m%b-%d%a'
	2016-W30-07Jul-27Wed
Time:
	date +'%I:%M %p'
	05:55 PM
	
tutorial: http://www.cyberciti.biz/faq/linux-unix-formatting-dates-for-display/
