====== Platform Target Definition ======
Created Tuesday 18 July 2017

To avoid duplicates in show call hierarchy, create 2 platform target definitions. One for "ALL", that will be used for platform development.
One for "SWT", that will be used only for swt development.

for swt, uncheck:
