====== listeners ======
Created Monday 30 January 2017

===== Lambda one-liners =====
widget.addListener(SWT.MouseDown, ev -> { Statment("..."); })

===== Short way =====
Use *adapters*:
		widget.addMouseListener(new MouseAdapter() {
			@Override
			public void mouseDown(MouseEvent e) {
				statment;
			}
		});
