====== disk commands ======
Created Saturday 17 March 2018

lsblk   #overview of partitions in compact form.
parted -l   # list partitions. (more human readable).
fdisk  -l   # list partitions. (bytes).

df -h  # shows mounted file systems. (-h human readable).
