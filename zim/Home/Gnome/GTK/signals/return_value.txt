====== return value ======
Created Tuesday 11 April 2017

In general:
true (1) - stop other handlers invoked for the event.
false (0) - to propagate the event further.
https://people.gnome.org/~gcampagna/docs/Gtk-3.0/Gtk.Widget-motion-notify-event.html
