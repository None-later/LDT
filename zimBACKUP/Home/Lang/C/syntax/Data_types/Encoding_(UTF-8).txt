====== Encoding (UTF-8) ======
Created Tuesday 25 October 2016

C often uses UTF-8 encoding. 

Pros:
- Ex Linux file names are in UTF 8
- It can represent all 1,114,112 Unicode characters. 

Cons:
- Can't access index in array (ex specific character).



http://www.cprogramming.com/tutorial/unicode.html
