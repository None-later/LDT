====== tr   (translate) ======
Created Tuesday 30 August 2016

Useful for string substitution.
ex:
pkg-config --cflags gtk+-3.0 | tr ' ' '\n'

 pkg-config --cflags gtk+-3.0 | tr ' ' '\n'
-pthread

-I/usr/include/at-spi2-atk/2.0
-I/usr/include/at-spi-2.0
-I/usr/include/dbus-1.0
-I/usr/lib64/dbus-1.0/include
