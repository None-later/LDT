====== cz17·08·18 visual block mode ======
Created Friday 18 August 2017
@blog @vim 

ctrl - V start visual block mode.
r - replace selected with character
c - change to something (like insert multiple text
d - delete selected
A - append after cursor
I - Insert in front of cursor.
o - move cursr to opposite corner
