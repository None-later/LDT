====== navigation ======
Created Friday 14 July 2017

Cursor:
gg   first line
G    last line        use  G$  to go to end of last line.
nG   nth line. e.g 12G

$  end of line 
0  start of line.

H  top of Screen
M  Middle of screen
L  Lower screen

Scroll:
z. Screen middle onto cursor
zt  screen top
zb  screen bottom.
C-y up a line
C-e down a line

Navigation:
C-D half page down
C-U half page up
C-F Forward page
C-B Backward page

Jumps/Searches:
 *   next whole word under cursor
 #   previous whole word under cursor 
 %   Jump to matching bracket.
