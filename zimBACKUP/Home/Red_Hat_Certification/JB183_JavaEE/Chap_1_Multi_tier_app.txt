====== Chap 1 Multi tier app ======
Created Wednesday 25 April 2018

JavaSE
- apps that run locally

**JavaEE**
- Build on top of JavaSE
- For Multi-threaded, transactional, secure, scalable applications
- Mostly API
	- Application server that passes "Technology Compatibility Kit (TCK)" for JavaEE is known as a JavaEE compliant application Server.

Notion of Profiles:
- subset of Api's
- e.g 
	- JavaEE Web Profile
	- JavaEE Full Profile

Red Hat JBoss Enterprise Application Platform (EAP) implements Web & Full Profile.
Implements API:
* Batch API
* Java API for JSON Processing (JSON-P)
* Concurrent Utilities
* WebSocket API
* Java Messaging Services (JMS)
* Java Persistence API (JPA)
* Java Connector Architecture (JCA)
* Java API for RESTful web services (JAX-RS)
* Java API for XML web services (JAX-WS)
* Servlet API
* Java Server Faces (JSF)
* Contexts and Dependency Injection )CDI)
* Java Transaction API (JTA)
* Enterprise Java Beans (EJB)
* Bean Validation API
