====== Bridging ======
Created Sunday 28 January 2018

Routing = Link networks, but have separate network.
Bridging = Link networks together into one network.

Repeaters = copy all signals.
Bridges = copy only relevant signals.

Types:
Simple = 2 networks
Multi-port = * networks
Transparent/Learning = Builds local DB, learns on-demand.
